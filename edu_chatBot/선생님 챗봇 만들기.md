## 4단계. 코드로 옮기기

2단계 표의 각 줄을 아래 형식 그대로 `RESPONSES` 배열에 옮기면 끝입니다.

```js
const RESPONSES = [
  { keywords: ['안녕', 'hi', '하이'], reply: '여기에 대답 입력' },
  { keywords: ['이름'], reply: '여기에 대답 입력' },
  // 표에 적은 줄 수만큼 계속 추가
];
```

`GREETING`과 `FALLBACK_REPLIES`도 3단계에서 적은 내용을 그대로 옮기면 됩니다.

### 왜 이 방식이 더 효율적인가?

- 챗GPT 같은 에이전트를 붙이면 과금을 해야 함
- 게임의 캐릭터 세계관, 취향, 목표 등을 만들 경우 작업 양을 많지만 코드에 직접 안 쓰이는 답변이 많아서 "쓴 내용"과 "코드에 넣을 내용" 사이에 변환 과정이 필요하므로 키워드 방식으로 설계를 진행하였음
- 이 설계표는 **표의 한 줄 = 코드의 한 줄**이라서, 학생이 표를 채우는 즉시 복사해서 붙여넣기만 하면 선생님 챗봇이 완성됨
- 총 작성량도 100문항 → 기본 8문항 + 키워드 12~14줄 수준으로 줄어들어 훨씬 수월함 
- 우리에게 시간 내에 수업이 끝나는 것은 중요하니까요~
