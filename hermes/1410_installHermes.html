헤르메스(Hermes) 에이전트 설치 가이드

24시간 돌아가는 나만의 자동화 에이전트를 텔레그램 위에 만들어 봅니다. Gemini API와 연동해 반복 업무를 대신 시키는 것이 목표입니다.

시작 전 준비물

  • 구글 계정 (Gemini API 키 발급용)
  • 텔레그램 앱 (봇 채널 생성 및 대화용)
  • Windows PowerShell (관리자 권한으로 실행)
헤르메스란 Claude Code vs Hermes 지침 파일 준비 무엇을 시킬 것인가 모델 참고 1단계 · API 키 발급 2단계 · 설치·환경설정 3단계 · 모델 설정 4단계 · 실행·테스트

개요

게이트웨이(Gateway)란?

OpenClaw나 Hermes 같은 AI 에이전트 프로그램이 텔레그램, 왓츠앱, 위챗 같은 메신저 앱과 실제로 연결되어 메시지를 주고받는 프로그램·서버 부분을 가리킵니다.

Nous Research에서 개발한 헤르메스 에이전트는 자가 개선 기능과 강력한 도구 사용 능력을 갖춘 오픈소스 에이전트 런타임입니다. 무료 티어인 Gemini API와 연동하고 텔레그램을 인터페이스로 사용하면 훌륭한 나만의 개인 비서를 구축할 수 있습니다.

사람이 할 수 있는 일을 대신 수행하면서 스스로 발전합니다. (돈을 많이 쓰면 잘 자랍니다.)

성장 사이클 작업 수행 → 학습 → 성장 → 반복

역할 분담

Claude Code vs Hermes

Claude Code로 소프트웨어를 개발하고, Hermes로는 반복적인 일을 시키면 좋습니다. 둘 다 사용하면 시너지가 납니다 — 사람은 관리만 하고, 반복적인 일은 에이전트가 합니다.

Hermes

  • 모닝 브리핑
  • 마케팅
  • 일정 관리

Claude Code

  • 코드베이스 개발
  • 커밋 관리

우선 이것부터

지침 파일 준비하기

Claude Code에 claude.md가 있듯, 헤르메스도 hermes.md 파일에 지침을 넣어둡니다. 비서, 마케터 등 원하는 페르소나를 이 파일에 정의합니다. 나에게 최적화된 에이전트를 만드는 과정이므로, 설치 전에 hermes.md 파일부터 준비하는 것이 좋습니다.

필요한 스킬(기능)은 skills.sh에 모아져 있으니 참고합니다.

기획

무엇을 시킬 것인가?

내 일상 업무 혹은 도움을 받을 용도로 사용하는 것이므로, 무엇을 시킬지와 어떤 역할을 맡길지를 먼저 정합니다.

예를 들어 오늘 특강에 오신 동추원몰 대표님을 마케팅 전문가로 설정했다면, 에이전트를 부르는 이름도 정해봅니다.

호출 예시 “동추원몰 대표님, 이것 좀 부탁드립니다.”

강사는 매일 은평경우회에 인사말을 보내고 있어서, 계절 + 간단한 교훈이나 글귀를 넣어 작성하는 포맷을 만들어 매일 안내하도록 설정한 예시가 있습니다.

참고

/model 선택 팁

OpenAI Codex 모델(GPT-5.5, 월 29,000원)이 추천됩니다. Claude는 6월부터 정책이 바뀌어 사용이 어렵고, 토큰이 매우 부족한 상황입니다.

준비가 끝나면 hermes setup으로 링크를 열어 로그인합니다.

1

사전 준비 — API 키 발급

Google Gemini API 키 발급

  • Google AI Studio에 접속하여 구글 계정으로 로그인
  • 'Get API key' 버튼을 클릭해 새 API 키 생성
  • 생성된 키(AIzaSy...)를 안전한 곳에 복사해 둠

텔레그램 봇 토큰 발급

  • 텔레그램에서 @BotFather를 검색해 대화 시작
  • /newbot 명령어 전송
  • 봇의 이름(Display Name)과 사용자 이름(Username, 반드시 bot으로 끝나야 함) 입력
  • BotFather가 발급한 HTTP API 토큰(123456789:ABC...) 복사
  • (선택) 그룹 메시지도 인식하려면 BotFather에게 /setprivacy를 보내 값을 off로 설정
참고 유저 ID는 텔레그램에서 @userinfobot을 통해 쉽게 확인할 수 있습니다.
2

Hermes Agent 설치 및 환경 설정

PowerShell을 관리자 권한으로 열고 아래 순서대로 진행합니다. (Windows 키 → PowerShell 선택)

공식 사이트: hermes-agent.nousresearch.com

설치

# Windows (PowerShell) 환경에서의 공식 설치 명령어입니다.
iex (irm https://hermes-agent.nousresearch.com/install.ps1)

환경 변수 구성

Hermes가 생성하는 설정 디렉터리(보통 ~/.hermes/.env) 또는 현재 환경 변수에 발급받은 키를 등록합니다.

# Gemini API 키 등록 (Hermes 네이티브 어댑터가 자동 인식)
export GEMINI_API_KEY="여러분의_구글_제미나이_API_키"

# 텔레그램 토큰 등록
export TELEGRAM_BOT_TOKEN="여러분의_텔레그램_봇_토큰"
보안 팁 본인의 텔레그램 계정 ID로만 접근을 제한하려면 .env 파일에 ALLOWED_TELEGRAM_USERS="본인_유저_ID" 설정을 추가하는 것을 권장합니다.
3

제공업체(Provider) 및 모델 설정

Hermes CLI를 실행하여 기본 언어 모델을 Gemini로 지정합니다. Hermes는 최신 Gemini 3 모델을 기본적으로 잘 지원합니다.

# 기본 제공업체를 gemini로 설정하고 모델을 지정합니다.
hermes model set gemini gemini-3-flash-preview
Tip 비용 효율성과 속도를 고려할 때 gemini-3-flash-preview나 gemini-1.5-flash 모델을 추천합니다.
4

게이트웨이 실행 및 테스트

모든 설정이 끝났다면 텔레그램 게이트웨이를 활성화하여 에이전트를 구동합니다.

hermes gateway start --platform telegram

서버가 정상적으로 구동되면, 스마트폰이나 PC의 텔레그램 앱에서 앞서 만든 봇 채널로 들어가 아무 메시지나 보내봅니다.

헤르메스 에이전트가 Gemini API를 사용해 답변을 생성하고, 파일 시스템 접근이나 웹 스크랩 등의 도구(Skills)가 필요할 때 스스로 판단하여 작업을 수행한 뒤 텔레그램 메시지로 결과를 보내줍니다.
