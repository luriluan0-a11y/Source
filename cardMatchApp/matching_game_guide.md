//8개의 단어와 설명

테스트 = 결함 식별 및 품질 평가를 위한 활동
디버깅 = 테스트/불만으로 발견된 결함 원인 추적.수정
결함 = 수행 결과와 예상 결과의 불일치
동적 테스팅 = 장애에서 결함, 결함에서 원인을 찾는 테스트 활동
테스트 케이스 = 입력과 예상 결과를 정의한 테스트 단위
검증(Verification) = 요구사항에 맞게 올바르게 만들어졌는지 확인
확인(Validation) = SW가 고객의 요구사항과 기대를 충족하는지 확인
정적 테스팅 = SW 실행 없이 코드, 문서, 설계 등을 검토하는 테스팅 기법, 주로 오류를 예방하거나 초기에 발견하기 위해 사용
