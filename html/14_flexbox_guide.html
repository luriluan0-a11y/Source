01

기본 개념

부모 요소(컨테이너)에 display: flex를 주면, 그 안의 자식 요소들이 자동으로 flex item이 되어 한 줄로 나란히 배치돼요. float처럼 직접 띄울 필요가 없어요.

기본 문법 
.container {
  display: flex;
}

컨테이너 안의 자식 요소들. 컨테이너의 설정에 따라 자동 배치돼요.

💡

주축(main axis)은 아이템이 나열되는 방향, 교차축(cross axis)은 그 반대 방향이에요. justify-content는 주축, align-items는 교차축을 담당해요.

06

실전 — div 3개 가운데 정렬

float을 쓰지 않고 flexbox로 div 3개를 가운데에 배치하는 방법이에요.

HTML 
<div class="all">
  <div class="div">...</div>
  <div class="div">...</div>
  <div class="div">...</div>
</div>
CSS 
.all {
  width: 1200px;
  margin: 0 auto;
  display: flex;
  justify-content: center;
  gap: 20px;
}

.div {
  /* float: left; 는 더 이상 필요 없음 */
  width: 350px;
}
💡

float: left는 항상 왼쪽부터 정렬돼서 가운데 배치가 어려워요. display: flex + justify-content: center를 쓰면 한 줄로 간단히 해결돼요.
