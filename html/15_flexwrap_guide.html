01

개요

flex 컨테이너는 기본적으로 자식 아이템을 한 줄에 모두 집어넣으려 해요. 아이템이 너무 많거나 넓으면 강제로 줄어들거나 컨테이너 밖으로 넘쳐요. flex-wrap을 사용하면 이 동작을 제어할 수 있어요.

문법
.container {
  display: flex;
  flex-wrap: nowrap;    /* 기본값 */
  flex-wrap: wrap;
  flex-wrap: wrap-reverse;
}
💡

flex-flow 단축 속성으로 flex-direction과 flex-wrap을 한 번에 쓸 수 있어요.
예: flex-flow: row wrap;

03

플레이그라운드

아이템 너비와 개수를 바꿔보면서 flex-wrap의 효과를 직접 확인해보세요.

.container {
  display: flex;
  flex-wrap: wrap;
  justify-content: flex-start;
  gap: 8px;
}
.item {
  width: 80px;
}

각 줄이 균등하게 늘어남

CSS
.container {
  display: flex;
  flex-wrap: wrap;          /* 줄이 여러 개여야 효과가 있음 */
  align-content: flex-start; /* flex-start | center | space-between | stretch */
  height: 300px;              /* 컨테이너 높이가 있어야 차이가 보임 */
}
💡

align-items는 한 줄 안에서 아이템들의 세로 정렬을 담당하고, align-content는 여러 줄 전체의 배치를 담당해요. 헷갈리기 쉬우니 꼭 구분해두세요!

# 카드 그리드 — 반응형 줄바꿈

화면이 좁아지면 카드가 자동으로 아래로 내려가는 반응형 레이아웃이에요.

Card 01설명 텍스트
Card 02설명 텍스트
Card 03설명 텍스트
Card 04설명 텍스트
CSS
.card-list {
  display: flex;
  flex-wrap: wrap;
  gap: 12px;
}

.card {
  flex: 1 1 120px; /* 최소 120px, 공간이 남으면 균등 확장 */
}
💡

flex: 1 1 120px에서 120px이 기준 너비예요. 이 값보다 공간이 부족하면 다음 줄로 내려가고, 남으면 균등하게 넓어져요. 이 값을 조절하면 한 줄에 들어오는 카드 수를 제어할 수 있어요.

요약

값 동작 언제 써요?
nowrap 한 줄 강제 유지 (기본값) 한 줄 레이아웃, 네비게이션 바
wrap 공간 부족 시 다음 줄로 카드 그리드, 태그 목록, 반응형 레이아웃
wrap-reverse wrap과 같되 줄 쌓임 방향 반대 채팅 메시지처럼 아래부터 쌓이는 UI
flex-flow 단축 속성
/* flex-direction + flex-wrap을 한 줄로 */
.container {
  flex-flow: row wrap;           /* 가로 방향 + 줄바꿈 */
  flex-flow: column wrap;        /* 세로 방향 + 줄바꿈 */
  flex-flow: row wrap-reverse;   /* 가로 방향 + 역방향 줄바꿈 */
}
