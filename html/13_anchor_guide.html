앵커 태그 <a>
완벽 가이드

웹 페이지의 링크를 만드는 핵심 태그. 외부 사이트, 내부 페이지, 이메일, 전화까지 모두 연결할 수 있어요.

목차

01. 기본 구조 02. 링크 유형 4가지 03. 주요 속성 04. 페이지 내 이동 05. 실전 예제 06. 주의사항
01

기본 구조

<a> 태그는 Anchor(닻)의 약자예요. href 속성에 이동할 주소를 넣고, 태그 사이에 링크로 보일 텍스트(또는 이미지)를 넣으면 돼요.

기본 문법 
<a href="이동할 주소">링크 텍스트</a>
예시 
<a href="https://google.com">구글 바로가기</a>

02

링크 유형 4가지

href에 어떤 값을 넣느냐에 따라 동작이 달라져요.

외부 링크

다른 웹사이트로 이동해요. https://로 시작해요.

code
<a href="https://naver.com"
   target="_blank">
  네이버
</a>

내부 링크

같은 사이트의 다른 HTML 파일로 이동해요.

code
<a href="about.html">
  소개 페이지
</a>

이메일 링크

클릭하면 메일 앱이 열려요. mailto:를 붙여요.

code
<a href="mailto:a@b.com">
  메일 보내기
</a>

전화 링크

모바일에서 클릭하면 전화 앱이 열려요. tel:을 붙여요.

code
<a href="tel:010-1234-5678">
  전화 걸기
</a>
03

주요 속성

<a> 태그에서 자주 쓰는 속성들이에요.

속성 설명 예시 값
href 이동할 주소 (필수) "https://..."
target 링크 열리는 위치 설정 "_blank" / "_self"
title 마우스 올리면 나오는 툴팁 "설명 텍스트"
download 클릭 시 파일 다운로드 download (값 없이)
rel 링크 관계 설명 (보안용) "noopener noreferrer"
target 속성 비교 
/* 새 탭에서 열기 */
<a href="https://google.com" target="_blank">구글</a>

/* 현재 탭에서 열기 (기본값) */
<a href="https://google.com" target="_self">구글</a>
04

페이지 내 이동

같은 페이지 안에서 특정 위치로 바로 이동할 수 있어요. #아이디명 형식으로 써요. 포트폴리오의 섹션 이동 버튼에 딱 맞아요!

구조 
/* 1단계: 링크 만들기 — href에 #아이디 */
<a href="#section1">섹션1로 이동</a>

/* 2단계: 이동할 목적지에 id 설정 */
<div id="section1">
  여기로 이동해요!
</div>
💡

부드러운 스크롤 효과를 주려면 CSS에 html { scroll-behavior: smooth; } 를 추가하면 돼요. 이 페이지 목차도 이 방식으로 만들었어요!

05

실전 예제

포트폴리오 페이지에서 바로 쓸 수 있는 코드예요.

포트폴리오 활용 예시 
/* 이메일 연락 버튼 */
<a href="mailto:xxxx@naver.com"
   title="이메일 보내기">
  연락하기
</a>

/* 포트폴리오 섹션으로 스크롤 */
<a href="#works">포트폴리오 보기</a>

/* GitHub 새 탭으로 열기 */
<a href="https://github.com/luriluan0-a11y"
   target="_blank"
   rel="noopener noreferrer">
  GitHub 바로가기
</a>

/* 파일 다운로드 링크 */
<a href="resume.pdf" download>
  이력서 다운로드
</a>
06

주의사항

보안: rel 속성 추가

target="_blank"를 쓸 때는 보안을 위해 rel="noopener noreferrer"도 함께 써줘야 안전해요.

링크 텍스트는 명확하게

"여기 클릭" 보다 "GitHub 바로가기" 처럼 어디로 이동하는지 알 수 있는 텍스트가 좋아요.

href 빠뜨리지 않기

href가 없으면 링크가 아닌 일반 텍스트처럼 동작해요. 반드시 href를 써줘야 해요.

#id는 정확히 일치해야 해요

페이지 내 이동에서 href의 #이름과 목적지의 id="이름"이 정확히 같아야 해요. 대소문자도 구별해요.
